# PtrLayout
A handy pull-to-refresh layout.

# Add repository
```gradle
repositories {
    maven {
        url 'https://dl.bintray.com/scausum/maven/'
    }
}
```

# Add dependency
```gradle
    compile 'com.scausum.ptrlayout:library:0.3.0'
```
